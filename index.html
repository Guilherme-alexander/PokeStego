📖 Documentação PokeStego

🚀 Início Rápido

1. Carregue o Corpus

Selecione uma geração de Pokémon ou use seu próprio corpus personalizado. O corpus é usado para treinar o modelo de linguagem.

// Selecionar corpus const corpus = "Pikachu usa Thunderbolt!";

2. Codifique

Digite sua mensagem secreta e clique em "Codificar". O texto gerado parecerá uma conversa normal sobre Pokémon.

// Codificar mensagem const encoded = stego.encode("Senha123");

3. Decodifique

Cole o texto codificado e clique em "Decodificar" para revelar a mensagem original.

// Decodificar const decoded = stego.decode(encodedText);

Parâmetro Descrição Valor Padrão Opções

punctuation Pontuação usada nas frases ['!', '?', '...'] Personalizável
delimiter Delimitador de sentenças '§|§' Qualquer string

💻 Exemplos de Código

JavaScript 
// Criar instância do PokeStego
const stego = new PokeStego();

// Usar configuração personalizada
const config = {
    nGramOrder: 4,
    punctuation: ['⚡', '🔥', '💧']
};
const stegoCustom = new PokeStego(config);

// Adicionar corpus personalizado
const novosPokemons = ['Pikachu', 'Charizard', 'Mewtwo'];
stegoCustom.addCorpusData(novosPokemons);

// Codificar mensagem
const mensagem = "A senha é 1234";
const codificado = stegoCustom.encode(mensagem);
console.log("Codificado:", codificado);

// Decodificar
const decodificado = stegoCustom.decode(codificado);
console.log("Decodificado:", decodificado);

// Ver estatísticas
const stats = stegoCustom.getStats();
console.log("Estatísticas:", stats);

📚 Referência da API

new PokeStego(config)

Cria uma nova instância do PokeStego com configurações opcionais.

encode(message)

Codifica uma mensagem usando esteganografia Pokémon.

decode(text)

Decodifica um texto codificado pelo PokeStego.

addCorpusData(data)

Adiciona novos dados ao corpus de treinamento.

getStats()

Retorna estatísticas do modelo atual.

buildModel(corpus)

Constrói o modelo n-gram a partir do corpus.
